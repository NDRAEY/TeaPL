# TeaPL

TeaPL - hobby programming language.

It's remake of CharmeleonPL

Currently, it transpiles code into C source.

# Features

- [x] Standard types
- [x] Variables
- [ ] Functions
- [ ] Control flow
- [ ] Loops
- [ ] String concatenation
- [ ] Overflow prevention
- [ ] Pointers
- [ ] Standard library
- [ ] FFI
- [ ] Dual Mode - With standard library and without standard library
- [ ] Memory safety
